# Figcoon

Figcoon is a tool designed to streamline the process of extracting and consolidating design elements from Figma. It helps in converting Figma variables, styles, JSON structures, and CSS properties into a unified and manageable format, making it easier for developers to implement designs consistently across various platforms.

## Features

- **Variable Consolidation**: Extract and organize design variables from Figma.
- **Style Management**: Consolidate text, color, and effect styles for consistent application.
- **JSON Structure Generation**: Convert Figma element properties into structured JSON for easy data transfer.
- **CSS Export**: Generate clean and organized CSS from Figma designs.

## Getting Started

To get started with Figcoon, follow these steps:

1. **Installation**: This repository currently contains a Figma plugin. To run it locally, clone the repository, navigate to the directory, and run `npm install` to install dependencies. Then, use `npm run build` to compile the TypeScript code.
   In Figma, go to `Plugins > Development > Import plugin from manifest...` and select the `manifest.json` file from this project.

2. **Usage**: Once installed, open a Figma file. You can then run the plugin from `Plugins > Development > Figcoon`. The plugin UI will appear, allowing you to extract and consolidate design elements.
